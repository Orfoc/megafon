Задание для разработчиков №2

Инструкция: 
0. Необходимо иметь библиотеки из стандартного набора. (tkinter, copy)
1. Запустить через интерпретатор python. (версии 3.6+)
2. Начальные условия вводятся посредством нажатия на конкретные клетки. (зеленая - заполнена, белая - пуста)
3. Игра начинается нажатием на кнопку "START GAME" последующие итерации кнопкой "NEXT GENERATION".
4. Нажав на кнопку "reset field" или "new game" сбрасывается поле до начального состояния, отменяется начатая игра.
Во время игры при нажатии на клетку покажется количество живых соседей.
